Cannonball Run Havoc Engine
System Attribution
The Havoc Engine used in this project was created by Rowan, Rook and Decard. It is a rules-lite, narrative-focused system originally developed for games such as Eat the Reich, Havoc Brigade, and Goat Crashers. Visit their website at rowanrookanddecard.com.

This implementation includes custom CSS and mechanical structures developed by Shawn Mohr, building upon the framework provided in the Havoc Engine Toolkit.

Overview
The Cannonball Run Havoc Engine is a high-octane, cinematic TTRPG. This repository contains the source code for the custom character sheet used in the Roll20 Virtual Tabletop.
Play Modes
Co-Op Mode: Players collaborate as a team to tackle scene OBJECTIVES and THREATS. GAS is treated as a shared team resource, allowing for tactical distribution among party members to fuel UNIQUE ABILITIES.

Competitive Mode: Players race against each other. GAS becomes an individual resource, and the player whose roll completes a scene objective dictates the group's route and earns a VICTORY TOKEN. The winner is determined by the most VICTORY TOKENS held at the end of the race.

Repository Structure
Cannonball Run Character Sheet Co-Op.html & Cannonball Run Character Sheet Co-Op.css: The structural and stylistic files for cooperative play.

Cannonball Run Character Sheet Competitive.html & Cannonball Run Character Sheet Competitive.css: The structural and stylistic files for competitive play.
